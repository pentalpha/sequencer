Algorithm for sequencing segments of text in order to return the full text.
Input - lines of text (%% = \n):
"Casa aba %% data
data live stream"

Output - full text:
"Casa aba
data live stream"

Algorithm made for the Bioinformatics Hackathon (http://bioinformatica.imd.ufrn.br/snnb/).
